# User story show bookmarked flights

When the user clicks this button the bookmarked flights will
be shown below one and other.

## Actors 

+ Application, User

## Input

Flight marked as favourite, Favourites table, Click event

## Internal state chage 

A new view appears that displays the bookmarked flights.

## Output

A new view with flights.

## Errors

+ Table not exists
+ No action executed
+ No view appears
+ Table empty (make a placeholder)
